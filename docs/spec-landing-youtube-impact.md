## 4. Comportements techniques

- **Marquees** : pause au hover, vitesse lente, `prefers-reduced-motion` respecté. Variante reverse pour rangées contre-sens.
- **Accordion FAQ + 5 étapes** : un item ouvert à la fois, premier ouvert par défaut.
- **Carrousel témoignage alumni** : flèches + dots, swipe mobile.
- **Glows** : CSS uniquement (radial-gradient + blur). Pas d'images lourdes.
- **Responsive** : mobile-first. Timeline horizontale → version verticale en mobile.
