## 2. Charte graphique

### Couleurs

| Rôle | Couleur | Hex |
|---|---|---|
| Fond sombre général | Violet très sombre | `#0f081e` |
| Fond clair général | Blanc cassé teinté violet | `#faf8fd` |
| Bande teintée intermédiaire | Violet très pâle | `#f1eafa` |
| Surface / dégradés | Violet | `#5d2999` |
| CTA, accents | Corail | `#ff605c` |
| Texte sur sombre | Blanc | `#FFFFFF` |
| Texte sur clair | Violet très sombre | `#1a1030` |
| Validation / checks | Vert | `#41af6e` |

Secondaires (touches ponctuelles) : bleu `#0f4696`, magenta `#ae08b7`,
jaune `#fff203`, vert lime `#a6ce13`.

### Signature visuelle

**Dégradé violet → corail** (`#5d2999` → `#ff605c`). Sur : mot clé du H1,
bordures carte pricing, fonds bannières CTA. Avec parcimonie.

### Typographie

- **Titres (H1-H3) :** Khand, condensé, 600-700.
- **Corps :** Inter, 400-500, interligne généreux.
- **Mot clé de chaque H1/H2 en italique** (signature typo du modèle).
  Exemples : *monétise*, *de zéro à ta chaîne monétisée*,
  *sous gestion Branddeo*, *les membres*, *le plus simple*, *tout inclus*.

### Principes design

**Rythme d'alternance fond sombre / fond clair** :
- **Sombre `#0f081e`** : barre annonce, navbar, hero, bandeau outils,
  FAQ + bloc contact, CTA final, footer.
- **Clair `#faf8fd`** : sections bénéfices, comment ça marche, pricing,
  témoignages, résultats, tableau comparatif.
- **Bande teintée `#f1eafa`** : barre micro-promesses qui sépare hero
  sombre des sections claires.

**Boutons :**
- Pill complet (border-radius max).
- CTA principal corail `#ff605c`, texte blanc, padding horizontal large.
- CTA fin de section très large (pleine largeur conteneur), flèche → finale.
- Sur dégradé : bouton blanc, texte `#0f081e`.

**Navbar :**
- Sombre `#0f081e`, sticky.
- **Barre de progression de scroll** : fin trait dégradé violet→corail
  sous la navbar.

**Cartes :**
- Sections claires : blanches, ombre douce, coins arrondis 16-20px,
  bordure quasi invisible.
- Sections sombres : `#1a1030` faible opacité, bordure fine violette.
- **Carte pricing — INVERSION DE CONTRASTE** : carte SOMBRE `#0f081e`
  sur section CLAIRE. Pattern clé du modèle.

**Checks et pastilles :**
- Checkmarks dans pastilles rondes vertes `#41af6e` (icône blanche).
- Listes niches : pastilles rondes dégradés palette devant chaque item.

## 4. Comportements techniques

- Smooth scroll vers `#tarif`, `#programme`, `#comment`, `#temoignages`, `#resultats`, `#faq`
- Barre annonce sticky refermable, état mémorisé session
- **Barre progression scroll** sous navbar (trait dégradé)
- Marquees : pause hover + `prefers-reduced-motion`
- FAQ accordéon premier item ouvert
- VSL embed/mp4 lazy
- Performance : webp/avif, lazy
- SEO : title "YouTube Impact — Lance, développe et monétise ta chaîne YouTube en 90 jours | Branddeo Academy"

## 5. Règles de rédaction

- Tutoiement, phrases courtes
- **Interdit :** "ce n'est pas X, c'est Y", motivationnel sur-produit, jargon vide, promesses chiffrées
- Promesse : méthode + accompagnement + communauté. Jamais revenus garantis.
